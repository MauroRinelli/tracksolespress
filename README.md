## Corrieri Supportati

- **UPS**: Codici che iniziano con "1Z"
- **DHL**: Numeri di 10-11 cifre
- **Poste Italiane**: Numeri di 13 cifre
- **FedEx**: Numeri di 12 o 14+ cifre

## Utilizzo

1. Apri `index.html` nel browser
2. Clicca su "Inizia il Tracking"
3. Compila i campi della tabella
4. Usa i pulsanti per tracciare, copiare o inviare
